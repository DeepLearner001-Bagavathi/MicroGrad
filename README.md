MicroGrad (AutoGrad) Implementation from Scratch

This project demonstrates the implementation of MicroGrad, a minimal automatic differentiation engine designed to compute gradients automatically for neural network training.
The engine builds a computational graph during the forward pass and performs backpropagation to calculate gradients efficiently.

The project includes:

Automatic differentiation using computational graphs
Backpropagation algorithm implementation
Gradient-based optimization
Neural network construction from scratch
Training and parameter updates using gradient descent
Educational examples to illustrate deep learning fundamentals

By recreating the core functionality of modern deep learning frameworks, this project helps developers and students gain a deeper understanding of how neural networks learn and how gradients drive the optimization process. 
It serves as an excellent learning resource for understanding the inner workings of machine learning and deep learning systems.
